Navigation

To create a navigation file, simply add it under the config folder. Example:

sidebar.yml
# Just a simple item
- label: 'Home'
  link: '/'
# With a list of items
- label: 'With subitems'
  items:
    - label: 'My Example'
      link: '/my-example'

You can also add external links:

- label: 'GitHub'
  link: 'https://github.com/jpedroschmitz'
