React란 무엇인가?

React는 페이스북에서 개발하고 관리하는 UI Library. Angular가 웹 애플리케이션 개발에 필요한 다수의 기능을 제공하는 것과는 대조적으로 React는 UI 기능만 제공한다. 따라서 전역 상태 관리, 라우팅, 빌드 시스템 등은 각 개발자가 직접 구축해야 한다. React팀에서는 진입 장벽을 낮추기 위해 개발환경을 구축해주는 create-react-app을 지원해준다.

React를 사용하는 이유는 무엇일까? 가장 큰 이유 중의 하나는 UI를 자동으로 업데이트해 준다는 점이다. 대개 프로그램의 상태가 변하면 UI도 변경되는데, 이는 다음과 같이 함축적으로 표현할 수 있다.
UI = render(state)
우리는 API통신이나 User Event를 통해서 프로그램의 상태값을 변경하고 React는 변경된 상태값을 기반으로 UI를 자동으로 업데이트한다. React는 Virtual Dom을 통해 이전 UI 상태를 메모리에 유지하고 변경될 UI의 최소 집합을 계산한다. 불필요한 UI 업데이트는 줄고, 성능은 좋아진다.
