### 벡터의 연산

|  **연산자 우선순위** | **뜻** | **표현방법** |
| **[ ]** | 인덱스 | a[1] |
| **$** | 요소 뽑아내기 | a$coef |
| **^** | 지수 | 5^2 |
| **+-** | 단항 마이너스, 플러스 | -3, +5 |
| **:** | 수열 생성 | 1:10 |
| **%any%** | 특수 연산자 | %/%나눗셈 몫, %% 나머지<br>, %*% 행렬의 곱 |
| <strong>* /</strong> | 곱하기 나누기 | 3*5, 3/5 |
| **+ -** | 더하기 빼기 | 3+5, 3-5 |
| **!** | Not | !(3==5) |
| **&** | and | TRUE & TRUE |
| <code style="color:black"><strong>|</strong></code> | OR | |
| **~** | 식(formula) | |
| **-> ->>** | 오른쪽 대입 | 3->a |
| **=**, **<- <<-** | 왼쪽 대입 | |
| **?** | 도움말 |
